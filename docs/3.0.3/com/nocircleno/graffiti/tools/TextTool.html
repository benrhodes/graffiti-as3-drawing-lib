Example
The following code creates an instance of the TextTool. 
        var fmt:TextFormat = new TextFormat();
        fmt.font = "Arial";
        fmt.size = 22;
        fmt.color = 0x000000;
        
        var textSettings:TextSettings = new TextSettings(fmt);
        
        var textTool:TextTool = new TextTool(textSettings);
